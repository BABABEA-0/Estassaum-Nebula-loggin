Nebula Station - Terminal de Acesso Espacial
Este projeto é um sistema de login temático desenvolvido para a disciplina de Interação Humano-Computador (IHC). O objetivo foi criar uma tela de login para uma estação espacial fictícia, priorizando a usabilidade e a imersão do usuário através de feedbacks visuais e sonoros.

O Nebula Station simula o portal de entrada de uma base espacial. Para tornar a experiência mais interessante, o sistema conta com um "protocolo de defesa" que é ativado caso um intruso tente acessar o sistema repetidamente com dados inválidos.


Tecnologias Utilizadas:
HTML5 & CSS3 - Para a estrutura e o visual "Glassmorphism" (efeito de vidro fosco).

JavaScript - Para a lógica de validação, animações e o sistema de som.

Web Audio - O sistema gera seus próprios sons (beeps e alarmes) sem precisar de arquivos externos, garantindo funcionamento offline.



Metas de Usabilidade:
Fácil de aprender/lembrar: O design segue o padrão universal de campos de login, tornando o uso intuitivo desde o primeiro contato.

Fácil de entender: O uso de ícones (como o olho para mostrar a senha e o "X" para erro) comunica estados do sistema sem a necessidade de textos complexos.

Útil: O sistema cumpre sua função de proteger o acesso ao terminal da estação.

Seguro: A opção de ocultar/exibir a senha e o feedback de "Acesso Negado" dão ao usuário o controle e a sensação de que o sistema é rigoroso.

Eficiente: Atalhos como a tecla "Enter" para logar agiliza a tarefa.


Metas de Experiência:
Esteticamente apreciável: Visual moderno com fundo sideral, estrelas e transparências.

Interessante: A narrativa de "invasor detectado" transforma uma tarefa comum em uma pequena experiência interativa.

Agradável: Cores equilibradas e sons suaves de "porta abrindo" ao ter sucesso no acesso.

Satisfatório: O feedback visual do cartão tremendo ao errar e o letreiro de sucesso trazem clareza e satisfação na conclusão.

Compensador: O usuário recebe uma saudação personalizada ao entrar, reforçando o sucesso da operação.


Como Testar:
Abra o arquivo index.html em qualquer navegador.

Para Acesso Permitido:

Usuário:(qualquer nome serve)

Senha: 1234

Para Protocolo de Expulsão:

Erre a senha propositalmente 3 vezes.

O sistema entrará em modo de alerta vermelho com contagem regressiva e som de alarme.


Funcionalidades Especiais
Mostrar Senha: Clique no ícone de olho para conferir o código digitado.

Feedback Sonoro: Sons gerados em tempo real para acertos e erros.

Sistema de Defesa: Sequência de "eliminação" do terminal caso ocorram múltiplas falhas de login.


Design Responsivo: O cartão de login se adapta ao tamanho da tela.
